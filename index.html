  • Smart Voice Assistant
  1. The Smart Voice Assistant developed is a desktop-based built application using python modules and libraries that can perform tasks or services for an individual based on voice commands or questions.
  2. When a user asks a personal assistant to perform a task, the natural language audio signal is converted into digital data that can be analyzed by the software. Then this data is compared with a database of the software using an innovative algorithm to find a suitable answer. - This smart voice assistant can perform personal task as well as Major task.
  3. The personal tasks that are supported by the smart voice assistant are: * Checking weather updates, Sending mails, Search on Wikipedia etc.
  4. The Major tasks that are supported by the smart voice assistant are: * Attendance system, COVID-19 updates of world and any country, Fire Detection and Alarm System, Surveillance System, etc.
  • IoT Home Automation
  1. First IOT project making a smart building with home automation using PIR sensors, LDR sensors and micro-controller (ArduinoUNO), another device known as ESP8266 to extract all the Information.
  2. PIR sensors senses the number of persons entered or left the room. - LDR sensor senses the amount of light in room so that if there is sufficient amount of light, bulb will not glow this information is extracted by ESP8266.
  3. All the data and information are visualized in ThingSpeak.
  4. This project has given me confidence to work further on Fire sensors, ultrasonic sensor, smoke sensor, ESP32, etc.
